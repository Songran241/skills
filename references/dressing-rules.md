# 穿搭法则

## 一、温度分层

以体感温度（结合湿度、风力修正后的温度）为基准：

| 体感温度 | 穿搭层级 | 核心单品 | 厚度要求 |
|----------|---------|---------|---------|
| < -5°C | 三层+：保暖内衣 + 厚毛衣 + 厚羽绒/派克大衣 | 加绒裤 + 高帮雪地靴/厚底靴 | 底层3、中层4、外层5 |
| -5~5°C | 三层：打底 + 毛衣/卫衣 + 羽绒服/厚大衣 | 长裤(可选加绒) + 靴子/厚底鞋 | 底层2-3、中层3-4、外层4-5 |
| 5~12°C | 两层半：长袖/薄毛衣 + 夹克/风衣/薄羽绒 | 长裤 + 运动鞋/皮鞋 | 底层2-3、外层3-4 |
| 12~18°C | 两层：T恤/衬衫 + 薄外套(牛仔/西装/针织开衫) | 长裤 + 板鞋/帆布鞋 | 底层2、外层2-3 |
| 18~25°C | 单层+：短袖/薄长袖，备薄外套 | 长裤/短裤均可 + 帆布鞋/船鞋 | 1-2 |
| 25~32°C | 单层：短袖/背心 | 短裤/薄长裤 + 凉鞋/网面鞋 | 1 |
| > 32°C | 极简单层：最轻薄款 | 短裤 + 凉鞋/拖鞋 | 1 |

## 二、天气修正

| 天气条件 | 外层要求 | 鞋履要求 | 面料建议 | 其他 |
|----------|---------|---------|---------|------|
| 降雨概率 > 50% | 防水/防泼水外套 | 防滑防水：靴子、皮革鞋 | — | 带伞，避免帆布鞋 |
| 降雨概率 > 80% | 专业防水外套 | 防水靴/橡胶底 | — | 避免长裤拖地 |
| 风速 > 5级 (≥8m/s) | 防风外套，避免宽松大衣 | 稳定鞋底 | 紧密编织面料 | 避免阔腿裤 |
| 湿度 > 80% + 高温 | 无需外层 | 透气网面 | 棉麻、羊毛（天然纤维） | 避免化纤贴身 |
| 紫外线指数 > 7 | 轻薄防晒外套/长袖 | — | — | 帽子/墨镜 |
| 早晚温差 > 10°C | 可叠穿，方便穿脱 | — | 洋葱式叠穿 | 早多一件，午脱 |

## 三、配色法则

### 3.1 色系定义

**中性色（万能基底）**
黑、白、灰、藏蓝、卡其、军绿、米白、深灰

**暖色调**
棕、驼、酒红、姜黄、橙色、土黄、砖红

**冷色调**
深蓝、灰蓝、墨绿、紫、靛蓝、雾蓝

### 3.2 三条铁律

1. **三色原则**：全身主色不超过 3 个（中性色不计数，鞋可按配饰色处理）
2. **明度拉开**：上下装明度需拉开——上深下浅或上浅下深，避免上下一体模糊
3. **单焦点**：亮色/花纹/logo/大面积印花单品一次只穿一件，其余用中性色压

### 3.3 推荐配色公式（按安全性排序）

**公式1: 同色系渐变** ★★★★★（最安全）
同色相、不同明度/饱和度做层次。
- 藏蓝衬衫 + 浅蓝牛仔裤 + 米白鞋
- 灰色毛衣 + 深灰西裤 + 白色鞋
- 卡其风衣 + 米白内搭 + 棕色鞋

**公式2: 中性色全幅** ★★★★★（闭眼不出错）
全身中性色，靠材质对比做层次。
- 白色T恤 + 黑色牛仔裤 + 灰色运动鞋
- 黑色高领 + 灰色西裤 + 黑色皮鞋

**公式3: 中性色 + 单点提亮** ★★★★
大面积中性色，一件亮色或高饱和单品做视觉中心。
- 黑T + 黑色裤 + **姜黄色**外套
- 全身藏蓝/灰 + **酒红色**鞋

**公式4: 冷暖对比** ★★★
一个冷色 + 一个暖色，形成张力但不冲突。
- 卡其风衣(暖) + 藏蓝内搭(冷) + 灰色裤(中性)
- 棕色外套(暖) + 灰蓝衬衫(冷) + 深蓝裤(冷)

**公式5: 大地色叠穿** ★★★
不同大地色系单品叠穿，暖感统一、层次丰富。
- 棕外套 + 米白内搭 + 深棕裤 + 卡其鞋
- 驼色大衣 + 燕麦色毛衣 + 巧克力色裤

### 3.4 配色避雷

| 雷区 | 原因 | 修复 |
|------|------|------|
| 深棕裤 + 黑色上衣 | 明度太近，腰线消失 | 深棕裤配米白/浅灰上衣 |
| 红 + 绿同饱和度 | 互补色对冲过强 | 降低一方的饱和度（酒红+墨绿可接受） |
| 全身超过 3 种鲜艳色 | 视觉碎片化 | 鲜艳色最多 1 个，其余用中性色 |
| 亮蓝 + 亮橙 | 运动品牌配色，日常难驾驭 | 降低明度或只保留一个亮色 |
| 同色同明度同质感 | 显臃肿无轮廓 | 至少拉开明度差或用不同面料 |

## 四、风格模板

### 4.1 极简通勤 (Minimalist Commute)
- **公式**：纯色T恤/衬衫 + 直筒西裤/深色牛仔裤 + 简洁皮鞋/小白鞋
- **色调**：黑白灰藏蓝，不超 2 色
- **面料**：棉 + 羊毛混纺，高级感优先
- **场合**：办公室、商务休闲
- **要点**：无 logo、无花纹、靠剪裁和面料说话

### 4.2 日系盐系 (Japanese Shio Style)
- **公式**：宽松衬衫/卫衣 + 阔腿裤/宽直筒裤 + 帆布鞋/德训鞋
- **色调**：低饱和度（灰、驼、米、雾蓝、鼠尾草绿）
- **面料**：棉麻、牛津纺、灯芯绒
- **场合**：日常出行、约会、咖啡馆
- **要点**：大一码选款、自然垂落、不追求腰线

### 4.3 City Boy
- **公式**：Oversized T恤/卫衣 + 短裤/宽松牛仔裤 + 厚底板鞋
- **色调**：基础色为主，允许一件亮色
- **面料**：重磅棉、毛圈布
- **场合**：逛街、周末、校园
- **要点**：叠穿领口外露、中筒袜在鞋口外、宽松但不邋遢

### 4.4 轻机能 (Techwear Light)
- **公式**：防水夹克/冲锋衣 + 工装裤/束脚裤 + 军靴/户外鞋
- **色调**：黑、军绿、灰、炭灰
- **面料**：尼龙、防撕裂、DWR涂层
- **场合**：雨天、户外、通勤防雨
- **要点**：功能性口袋、拉链细节、裤脚可调节

### 4.5 简约精致 (Smart Casual)
- **公式**：针织衫/Polo/衬衫 + 斜纹裤/西裤 + 乐福鞋/德比鞋
- **色调**：藏蓝、卡其、灰、棕
- **面料**：美利奴羊毛、棉麻混纺
- **场合**：约会、半正式聚会、精致日常
- **要点**：上衣扎进裤子或法式塞、皮带与鞋同色

### 4.6 运动休闲 (Athleisure)
- **公式**：运动T恤/卫衣 + 慢跑裤/运动短裤 + 跑鞋/复古球鞋
- **色调**：黑白灰 + 一个品牌色
- **面料**：速干、弹力
- **场合**：运动、周末闲逛、取快递
- **要点**：合身不紧身、避免全身同品牌 logo

## 五、风格一致性

定义风格模板与鞋履、外层的匹配规则。单品可以好，但频道不在一起就打架。

### 5.1 风格 × 鞋履匹配矩阵

| 风格 | ✅ 推荐鞋款 | ❌ 避免鞋款 | 原因 |
|------|-----------|-----------|------|
| 极简通勤 | sneakers(简洁)、loafers | boots(厚重)、sandals | 精致感优先，厚重靴破坏线条 |
| 日系盐系 | sneakers(帆布/德训)、loafers | boots、sandals | 低饱和松弛感，帆布/麂皮最对 |
| City Boy | sneakers(厚底板鞋) | loafers、sandals | 厚底是关键，乐福太精致 |
| 轻机能 | boots、sneakers(户外/复古跑鞋) | loafers、sandals | 户外功能感，乐福完全不对频道 |
| 简约精致 | loafers、sneakers(皮质小白鞋) | boots、sandals | 乐福/德比鞋是核心 |
| 运动休闲 | sneakers(跑鞋/复古球鞋) | loafers、boots | 运动感纯粹，皮鞋类全避 |

### 5.2 风格 × 外层匹配矩阵

| 风格 | ✅ 推荐外层 | ❌ 避免外层 | 原因 |
|------|-----------|-----------|------|
| 极简通勤 | blazer、coat、轻薄夹克 | hoodie(连帽) | 连帽自带的运动感破坏极简 |
| 日系盐系 | coat(风衣)、blazer(宽松)、无外层 | jacket(战术/机能) | 松弛感排斥硬核功能 |
| City Boy | hoodie、sweatshirt、宽松夹克 | blazer、coat | 街头感拒绝正式感 |
| 轻机能 | jacket(防水)、vest、parka | blazer、coat(羊毛) | 功能性唯一标准 |
| 简约精致 | blazer、coat、针织开衫 | hoodie、运动夹克 | 精致与运动对立 |
| 运动休闲 | hoodie、运动夹克、无外层 | blazer、coat | 运动感纯粹 |

### 5.3 风格冲突常见雷区

| 雷区 | 冲突点 | 修复 |
|------|--------|------|
| 软壳冲锋衣 + 乐福鞋 | 机能外 vs 精致脚，频道打架 | 鞋换复古跑鞋/军靴（如 P-6000 / 登山鞋） |
| 连帽卫衣 + 皮鞋/德比鞋 | 街头感 vs 正装感 | 鞋换板鞋/帆布鞋；或上装换针织衫/衬衫 |
| 西装外套 + 运动短裤 | 正式上 vs 运动下 | 裤换斜纹短裤/牛仔裤 |
| 运动服 + 乐福鞋 | 功能运动 vs 优雅休闲 | 鞋换跑鞋/复古球鞋 |
| 全身机能(软壳+工装裤+军靴) + 精致配饰 | 粗犷 vs 精细 | 配饰也走户外风，或弱化机能单品 |

### 5.4 叠穿跨风格混搭原则

跨风格混搭可以成立，但需要**一主一从**：

- **风格主调占 60%**：两件以上单品同风格 → 确定基调
- **从属风格占 40%**：一件单品跨风格 → 制造张力但不翻车

✅ 可接受的跨风格示例：
- 轻机能软壳衣 + 日系盐系阔腿裤 + 帆布鞋 → 上机能下盐系，鞋向裤靠拢
- City Boy卫衣 + Smart Casual乐福鞋 → 街头基底、精致点缀

❌ 翻车示例：
- 轻机能软壳衣 + Smart Casual乐福鞋 → 两个风格对撞，没有主从，纯打架
- 极简衬衫 + 运动跑鞋 + 工装裤 → 三件三个频道，碎片化

## 六、叠穿法则

适用于 5~18°C 或早晚温差大的场景：

1. **内层贴身**：最薄、透气（T恤/背心/秋衣）
2. **中层保暖**：毛衣/卫衣/法兰绒衬衫
3. **外层防护**：夹克/风衣/大衣/羽绒

**叠穿配色要点**：
- 内浅外深 → 稳重大方
- 内白/灰/藏蓝做安全内搭 → 百搭一切外层
- 领口叠穿（T恤领 + 衬衫领 + 外套领）时，最内层领口要低于中层

## 七、场景自适应

| 场景 | 风格推荐 | 注意 |
|------|---------|------|
| 上班/办公室 | 极简通勤 / Smart Casual | 避免大面积印花、破洞 |
| 雨天通勤 | 轻机能 / 极简通勤+防水外套 | 鞋防水、裤脚不拖地 |
| 周末出街 | City Boy / 日系盐系 | 放松但不随便 |
| 约会 | Smart Casual | 精致但不刻意，避免运动装 |
| 户外/徒步 | 轻机能 | 功能性优先，叠穿便于调节 |
| 正式场合 | 极简通勤/正装方向 | 衬衫+西裤+皮鞋，配色保守 |
