## 保暖能力 (warmth)

跨品类的绝对保暖值，1~5。同一件薄款羽绒服和一件厚卫衣可能保暖值相近，但物理体积完全不同（见 bulk）。

| warmth | 标签 | 典型单品（跨品类参考） |
|--------|------|----------------------|
| 1 | 无保暖 | 薄T恤、亚麻衬衫、网面鞋、短裤 |
| 2 | 微保暖 | 常规棉T、薄长袖、衬衫、牛仔裤、普通运动鞋 |
| 3 | 轻保暖 | 卫衣、法兰绒衬衫、**薄款羽绒服**、加绒裤 |
| 4 | 中保暖 | 厚毛衣、摇粒绒、常规羽绒服、加绒靴 |
| 5 | 强保暖 | 厚羽绒服、派克大衣、羊毛大衣、雪地靴 |

**warmth 推断规则**（从材质/描述推断）：
- 网面、亚麻、薄棉 → warmth 1
- 常规棉、单层、无填充 → warmth 2
- 加绒、法兰绒、薄填充（轻羽绒/抓绒）→ warmth 3
- 厚填充、厚针织、羊毛 → warmth 4
- 高克重填充、派克、厚羊毛 → warmth 5
- 羽绒服：warmth 最低从 3 起（再薄也有填充物保暖）

## 物理体积 (bulk)

同品类内的相对体积，1~3。用于判断叠穿兼容性——保暖强但体积小的单品（如薄羽绒）可以穿在修身外套内，体积大的不行。

| bulk | 标签 | 典型单品 | 叠穿兼容 |
|------|------|---------|---------|
| 1 | 轻薄 | 薄T恤、衬衫、薄打底、**薄款羽绒服** | 可穿在任何外层下 |
| 2 | 适中 | 常规卫衣、软壳衣、牛仔裤、运动鞋 | 可穿在常规/宽松外层下 |
| 3 | 厚重 | 厚毛衣、厚羽绒、大衣、靴子 | 仅适合宽松外层或作最外层 |

**叠穿兼容矩阵**（内搭 bulk × 外层 fit）：

| 内搭 bulk \ 外层 fit | 修身 | 常规 | 宽松 |
|---------------------|:---:|:---:|:---:|
| 1 轻薄 | ✅ | ✅ | ✅ |
| 2 适中 | ❌ | ✅ | ✅ |
| 3 厚重 | ❌ | ❌ | ✅ |

**bulk 推断规则**：
- 单层面料、可折叠很小 → bulk 1
- 常规厚度、有衬里 → bulk 2
- 高克重填充、厚针织、多层结构 → bulk 3
- 羽绒服：薄款 bulk=1，常规 bulk=2，厚款 bulk=3

## 面料识别要点

- **棉**: 哑光、柔软、有纹理 — 观察表面是否有细密织纹
- **羊毛/羊绒**: 表面有短绒毛感、针织纹理明显
- **亚麻**: 有明显粗细不一的纱线纹理、易皱
- **化纤/尼龙**: 有光泽、平滑、常见于功能外套
- **牛仔**: 斜纹明显、靛蓝色居多、有洗水痕迹
- **皮质**: 光泽柔和、有毛孔纹理

## 版型 (fit)

| 类别 | 识别要点 |
|------|---------|
| 常规 | 不故意宽不故意窄，肩线落在肩膀 |
| 宽松/Oversize | 肩线下落、衣身阔、袖笼大 |
| 修身 | 有明显腰线收窄 |
| 直筒 | 从臀部到裤脚宽度一致 |
| 阔腿 | 裤管明显宽大 |
| 束脚 | 裤脚有罗纹收口 |

## 风格 (style)

| 标签 | 特征 |
|------|------|
| casual | 日常休闲，无明显风格指向 |
| minimalist | 纯色、无logo、简洁线条 |
| formal | 正装/商务，衬衫西装类 |
| sporty | 运动品牌、功能性面料 |
| streetwear | 有大logo、印花、宽松剪裁 |
| workwear | 工装感、机能口袋、厚实面料 |
